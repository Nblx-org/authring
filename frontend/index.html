Authring

The Soul of The New Machine is You.

Authring is a code regulator: an AI API interceptor that audits and controls agentic code modification proposals from any agent to any model. By using historical impact, contributor expertise, and production risk assessment, it can invalidate agentic code changes chunk by chunk and reroute decisions at a fine granularity. The system utilizes:

  • Astra DB for historical data storage
  • Unstructured.io for document parsing
  • Langflow for AI-driven decision-making
  • Twilio for real-time human validation
  • GitHub CLI (gh) for managing patch review links
